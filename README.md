# library-borrowing-assistant
Console-based Python system for managing a single library member’s borrowed items, due dates, and return status.

## Features
- Store member personal details
- Add borrowed items
- View all borrowed items
- Group items by due date
- Mark items as returned

## Technologies Used
- Python 3
- Lists and Dictionaries
- While loops
- Conditional statements

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Run:

```bash
python main.py
